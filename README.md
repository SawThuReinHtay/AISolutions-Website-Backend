# AI Solutions Website Backend

This repository contains the **backend code** for the AI Solutions Website, a full-stack dynamic web application for a fictional AI company.  

The backend is responsible for **handling server-side logic, API endpoints, database interactions, authentication, and admin functionalities**.

---

## Project Overview

The backend supports the following core functionalities:

- User authentication and role-based access control (Admin / Staff)  
- Contact query management (view, update status, delete, search)  
- User management (add, edit, delete, search)  
- Blog management (create, edit, delete blogs with images)  
- Solutions management (add, edit, delete solutions)  
- Gallery management (upload, edit, delete images)  
- Reviews management (add, edit, delete reviews, category handling)  
- AI chatbot integration (handles requests and responses via external AI API)  

The backend communicates with a **MySQL database** to store and manage all application data.

---

## Technology Stack

- **Language:** Python  
- **Framework:** Flask  
- **Database:** MySQL  
- **Other Tools:** MySQL Connector, REST API design, Postman (for testing)
